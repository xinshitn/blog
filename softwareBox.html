软件盒子

我常用的一些软件，放这里既方便自己下载，也方便分享，嘿嘿

PAW
Paw HTTP Client for mac是一款Mac上的HTTP客户端模拟测试工具，可以让Web开发者设置各种请求Header和参数，模拟发送HTTP请求，测试响应数据，支持OAuth, HTTP Basic Auth, Cookies等，这对于开发Web服务的应用很有帮助，非常实用的一款Web开发辅助工具。 MAC 2.2.2

Charles Proxy
其通常称为Charles，是目前最强大的http调试工具，在界面和功能上远强于Fiddler，同时是全平台支持，堪称圣杯级工具，除了支持外部代理以外，还支持端口映射，反向代理，还有个访问控制，连接他需要输入用户名和密码，这成了标准的代理服务器了都，还能模拟慢网速等。 MAC 3.10.2

JD-GUI
自从2010年0.3.3版本后，该工具终于迎来了一次更新。JD-GUI是一个独立图形界面的Java源代码“.class”文件反编译工具。可以浏览重建的源代码，JD-GUI的即时访问方法和字段。JD-GUI是免费的非商业用途。其实现了ecliPSe的高亮着色功能，多文件标签浏览。注:Win版为0.3.6版 MAC WIN 0.3.5

SourceTree
Windows和Mac下免费的 Git 和 Hg 客户端，同时也是Mercurial和Subversion版本控制系统工具。支持创建、克隆、提交、push、pull 和合并等操作。其拥有一个精美简洁的界面，大大简化了开发者与代码库之间的Git操作方式，这对于那些不熟悉Git命令的开发者来说非常实用。注:Win版为1.7.0版 MAC WIN 2.1

Ngrok
在开发项目时，经常会遇到需要外网方法测试的情况。比如微信后台测试，这时候就很麻烦了，启用Ngrok，秒秒钟解决问题。个人建议使用1.7版，因为ngrok官方服务器在国外，你懂得，而最新版不支持自定义服务器，不知道付费内容行不行，总之1.7版已经足够我们用了，下载包中已经写好配置和示例了。 MAC & WIN 1.7
